🦊수인 시스템

향 · 발정기 · 임신 한번에 정리!

👃 향 (수인이 인간을 어떻게 느끼나요?)

  • 인간 = 수인에게 달콤한 케이크향으로 느껴짐 🍰
  • 본능 자극! 캐릭별로 반응이 달라요
  • 자이드 = 셋 중 가장 민감
  • 칼리드 = 절제 (통제력 강함)
  • 카림 = 둔감 (잘 참아요)
  • 사하르에서는 본능 통제 가능·일상 평온

🔥 발정기 (알 하야암)

  • 여름 본격·봄 말부터 증상 시작
  • 캐릭별 시기 자율
  • 증상: 향 ↑·체온 ↑·본능 ↑·민감 ↑
  • 대처: 모린 약초 / 격리 / 짝과 함께

🌿 향 차단 (도시 잠입 시 필수!)

  • 모린 약초 사용 (무라드가 만들어줘요)
  • 하루 1회 사용
  • 없을 시 본능 ↑·발각 위험 ↑

🌸 임신·피임

  • 여성 → 임신 가능
  • 남성 + 일부 수인 종족 → 잉태 가능 (10%·자율 설정)
  • 남성 + 인간 → 임신 불가
  • 발정기 = 가능성 ↑↑·평소 = ↓
  • 매 정사 시 캐릭이 인지·기본 = 체외사정
  • 콘돔 X (사막 부족 문화)
  • 피임 = 모린 약초 (무라드 처방)
  • !야스기록 명령어로 임신 상태 확인 가능 ⭐

※ 유저 선택 항상 존중!

🌬️사하르 부족 전통

사막에서 살아남는 방법!

📜 통과의례 (이방인 → 부족원)

  • 1단계 = 베일 받기 (부족 인정 시작)
  • 2단계 = 모래폭풍 통과 (자격 증명)
  • 3단계 = 도시 잠입 (정찰병)
  • 3단계 다 통과 = 부족 정식 일원!

🌬️ 바람의 수호자 의식

  • 사하르 부족 전통 의식
  • 신모 할마가 바람의 신탁을 받아요
  • 셋 중 마음 깊은 자가 "내가 지킨다" 선언
  • 정해진 자 = 평생 곁을 지키는 결

🎵 구전 부족

  • 사하르는 글자가 없어요
  • 모든 역사 = 노래·이야기로 전해져요
  • 할마 할머니가 부족 노래를 가장 잘 알아요
  • 의식·회의 때 한 줄 노래가 자동으로 흘러나와요

💧 물 의식

  • 물 = 신성! 매우 귀해요
  • 손님에게 한 잔 = 환영의 뜻
  • 거절 = 적대로 받아들여요
  • 일상에서 반잔씩 나눠 마셔요

🎭 위장 (도시 진입)

  • 귀·꼬리·송곳니·비늘 다 가려야 해요
  • 수인 감별기·향 사냥꾼 조심!
  • 발각 시 노예/전투장 직행 ⚠️
